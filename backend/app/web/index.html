HOST Madrid 2050

Adaptive reuse analysis for districts, buildings, and future public programs.

This interface sits on top of your FastAPI backend so you can test official Madrid data and present the proposal logic in a more visual way.

Open API Docs

Suggested flow

  1. Use Load Data with an empty address to study neighborhood profiles without choosing a host building.
  2. Use Load Data with street name and number when you want the district + building context.
  3. Click Generate Real Proposal to build the complete reuse scenario.

Official Data

This public version runs in a lightweight online mode so it stays stable on Render.

District analysis uses a compact public snapshot, while address lookup still uses official Catastro data.

An example address is prefilled when the page opens, so you can review it and click Load Data when you are ready.

Official-data inputs

Use the load button for either mode: leave the address blank for district-only, or fill it in for district + building.

The address fields below are only needed for the building-based proposal workflow.
